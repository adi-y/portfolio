# Aditya Yadav Portfolio

A modern portfolio website built with React and Vite to showcase my projects, technical skills, and software engineering journey.

---

## About

This portfolio serves as a central hub for my work, featuring selected projects, technical expertise, and contact information. The goal is to present my experience in backend development, distributed systems, and AI-powered applications through a clean and interactive interface.

---

## Features

- Responsive design for desktop and mobile
- Interactive landing page
- Smooth navigation between sections
- Project showcase with GitHub repositories
- Contact form powered by EmailJS
- Modern and lightweight user experience

---

## Tech Stack

**Frontend**

- React
- Vite
- CSS

**Libraries**

- tsParticles
- React Icons
- EmailJS

---

## Sections

### Home

Introduction and personal branding.

### About

Background, education, and technical skills.

### Projects

A collection of projects with descriptions and source code links.

### Contact

Direct communication through an integrated contact form.

## Getting Started

Clone the repository:

```bash
git clone https://github.com/adi-y/portfolio.git
cd portfolio
```

Install dependencies:

```bash
npm install
```

Start the development server:

```bash
npm run dev
```

Build for production:

```bash
npm run build
```

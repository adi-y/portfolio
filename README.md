# Aditya Yadav — Portfolio

Personal portfolio website built with React + Vite.

## Tech Stack

- React
- Vite
- react-tsparticles
- react-icons
- EmailJS

## Sections

- **Home** — Animated particle background with intro
- **About** — Background and tech stack
- **Projects** — Showcase of projects with GitHub links
- **Contact** — EmailJS powered contact form

## Run Locally

```bash
npm install
npm run dev
```

deployed on vercel
